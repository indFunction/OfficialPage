　某企画にて試合を管理するソフトウェアで効率化を図ろうという案が挙がったのですが、需要が少ないせいか満足できる代物をインターネットの波から見つけることができず、結果的に自作する羽目になりました。現代的なデザインと私のエゴが融合したグラフィック重視のソフトウェア「**Play on Display**」の始まりです。

<ImageLink href='https://github.com/indFunction/PlayOnDisplay/releases' src='https://raw.githubusercontent.com/indFunction/PlayOnDisplay/main/Assets/Images/Hero/Hero%20(Type%20B%2C%20FHD).png' />

## 概要

　本ソフトウェアでは三次元空間を用いてトーナメント表の作成、並びに画像や文章を複数の（仮想）ディスプレイ単位で表示・管理することができます。円形に並べられたディスプレイを往来することで多くの情報が取り扱え、当然ながらデータの入力や出力に対応しています。

## 機能

　ディスプレイを作成した際に、取り扱う媒体（メディア）を選択します。ボタンを押すことで画面が切り替わります。

<PictureGallery data={['/media/images/0011/0001.png']} border />

### 試合；勝ち抜き戦

　最大人数100人までのトーナメント表を自動的に生成します。3人以上の対戦人数の設定も可能で、セットアップ画面では専用のCSVファイル（本ソフトウェアで出力したデータ）の他にテキストファイル（TXT形式）から名簿を読み取る機能もあります。

<PictureGallery data={['/media/images/0011/0002.png']} border />

　メイン画面ではグラフィカルなトーナメント表が表示されます。プレイヤーのアイコンを選択し、名前や色の編集、勝敗の確定を行うことができます。左上に試合の情報、右上にデータの入出力や操作を行うボタンが設置されています。

<PictureGallery data={[
    '/media/images/0011/0003.png',
    '/media/images/0011/0004.png'
]} border />

### 試合；総当たり戦

　最大人数50人までのリーグ表を自動的に生成します。人数と試合の方式（点数で決めるか、勝敗で決めるか）が決められ、同様にデータの読み込みにも対応しています。

<PictureGallery data={['/media/images/0011/0005.png']} border />

　リーグ表ではプレイヤーや勝敗の編集内容が自動的に対称化されます。他にも、勝敗の方式や得点の比重を即時に変更することや、対戦の順序を出力することも可能です。

### 画像

　コンピューター内に保存されている（PNG形式もしくはJPG形式の）画像ファイルを表示します。どのような縦横比でも画面内に収まりますが、動作環境や画像ファイルの大きさ（解像度）によっては読み込みに時間が掛かります。ヒーロー画像の掲示に如何でしょうか。

<PictureGallery data={['/media/images/0011/0010.png']} border />

### 文章

　文章を書き込むためのフィールドを表示します。画面上で直接テキストを編集するか、右上のボタンからテキストファイル（TXT形式）の入出力も可能です。リッチテキストには対応していません。スケジュールの表示などに活用できます。

<PictureGallery data={['/media/images/0011/0011.png']} border />

## 操作

　基本的にマウスを用いて操作します。一部の操作はキーボードから行うこともできます。出来るかぎり直感的な操作を追求したので、おそらく初見でも扱えると思います。申し訳ありませんが、ゲームコントローラーには対応していません。

### コマンド操作

　右上の青いボタンからディスプレイの作成、赤いボタンからディスプレイの削除を行います。また、左右中央に設置されたボタンからディスプレイの移動と配置変更を行います。配置変更に関しては、現在の地点にあるディスプレイと矢印の方向にあるディスプレイを入れ替えるような感覚です。

<PictureGallery data={['/media/images/0011/0012.gif']} border />

　前面のインターフェースにあるボタンはホバーすることでコマンドの内容とショートカットキーが表示されます。ショートカットキーの一覧については以下の表を参考にお願いします。

| キー | コマンド |
| :--- | :--- |
| `[F1]` | インターフェースの表示・非表示を切り替えます。 |
| `[Ctrl]` + `[A]` | ディスプレイを追加します。 |
| `[Ctrl]` + `[R]` | ディスプレイを削除します。 |
| `[Left Arrow]` | 前のディスプレイへ移動します。 |
| `[Right Arrow]` | 次のディスプレイへ移動します。 |
| `[Ctrl]` + `[Left Arrow]` | ディスプレイを前のディスプレイと入れ替えます。 |
| `[Ctrl]` + `[Right Arrow]` | ディスプレイを次のディスプレイと入れ替えます。 |
| `[Ctrl]` + `[Q]` | ソフトウェアを終了します。 |
| `[Y]` | 警告ポップアップの「許可」を実行します。 |
| `[N]` | 警告ポップアップの「取消」を実行します。 |
| `[Enter]` | エラーポップアップの「了解」を実行します。 |

### カメラ操作

　全てのディスプレイでは画面の拡大や縮小、移動が行えます。ズーム（拡大・縮小）はスクロール、ズーム初期化はホイールクリック、パン（移動）は右クリックを押し続けた状態でカーソルを移動します。

<PictureGallery data={['/media/images/0011/0013.gif']} border />

　画面の操作については漏れなくキーボードからも行えます。マウスと併せて、操作内容については以下の通りです。一部、ショートカットキーのみから実行可能なコマンドが存在するのでご注意ください。

| キーボード | マウス | コマンド |
| :--- | :--- | :--- |
| `[W]` `[A]` `[S]` `[D]` | `[Right Button]` + `[Pad]` | 画面をパン（移動）します。 |
| `[Q]` `[W]` | `[Scroll]` | 画面をズーム（拡大・縮小）します。 |
| `[-]` | `[Middle Button]` | ズーム（拡大・縮小）を初期化します。 |
| `[Z]` |  | ディスプレイ全体を表示／切り替えます。 |
| `[F]` |  | 試合が表示されているディスプレイにて、次の対戦を表示します。 |

## 推奨環境

　インテル® Celeron® N4120のような省電力CPU／GPUでも60FPS程度は出たので大丈夫だと思いますが、推奨する動作環境を以下に記しておきます。

| 項目 | 推奨性能 |
| :--- | :--- |
| OS | Windows 7以上 |
| CPU | 第4世代／インテル® Core™ i3以上 |
| GPU | NVIDIA GeForce® GTX 980以上 |
| RAM | 2GB以上 |
| ROM | 100MB以上 |
