　大学や現場からGitを操作するとき、もしくは特殊な場所にGitサーバー（リポジトリー）がある場合は、プロキシーを経由しなければなりません。その場合は、以下のコマンドでプロキシーを設定します。

<Code type='Bash'>
&#036; git config --global http.proxy http://<ST type={['yellow_tc']}>&lt;プロキシーアドレス&gt;</ST>:<ST type={['yellow_tc']}>&lt;ポート番号&gt;</ST>

<ST type={['gray_tc']}>例：金沢工業大学の場合</ST>
&#036; git config --global http.proxy http://<ST type={['yellow_tc']}>wwwproxy.kanazawa-it.ac.jp</ST>:<ST type={['yellow_tc']}>8080</ST>
</Code>

<br />

　続いて、設定を確認しましょう。以下のコマンドの結果に含まれる`http.proxy`が指定したアドレスであれば成功です。

<Code type='Bash'>
&#036; git config --global -l
</Code>

<br />

　プロキシーの設定を元に戻す（削除する）場合は、以下のコマンドを実行します。コンフィグから`http.proxy`の項目が消えていれば成功です。

<Code type='Bash'>
&#036; git config --global --unset http.proxy
</Code>
